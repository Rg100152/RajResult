# 🎓 RajResult — Smart University Result & Academic Records Portal

**RajResult** is a premium, frameworkless Academic Result Management Portal built for universities and colleges. It allows students to securely view semester results, track SGPA/CGPA, explore academic analytics, and download printable marksheets — all through a sleek glassmorphism dashboard with smooth animations.

---

## ✨ Key Features

- **🔐 Student Authentication** — Login with enrollment number & password, or register a new account.
- **📊 Dynamic Dashboard** — Latest SGPA/CGPA, attendance, and quick action buttons.
- **🔍 Result Search** — Find results by enrollment number and semester with instant loading.
- **📄 Subject‑wise Marksheet** — Detailed table with internal/external marks, grades, and total.
- **📈 Academic Analytics** — Canvas‑drawn bar chart visualizing SGPA progression across semesters.
- **📋 Previous Results** — Semester‑wise record with SGPA/CGPA comparison.
- **🔔 Notifications** — Latest exam schedules, result declarations, and university announcements.
- **👤 Student Profile** — Personal and academic details in one place.
- **📱 Fully Responsive** — Optimized for desktop, tablet, and mobile.
- **🎨 Premium Dark UI** — Glassmorphism cards, neon gradients, floating particles, and smooth page transitions.
- **💾 LocalStorage Demo Data** — Pre‑loaded with sample student records for instant testing.

## 🛠️ Technology Stack

| Layer        | Technology                         |
| ------------ | ---------------------------------- |
| **Markup**   | Semantic HTML5                     |
| **Styling**  | CSS3 (Custom Properties, GPU Composites) |
| **Scripting**| Vanilla JavaScript (ES6+)          |
| **Data**     | Browser `localStorage`             |
| **Charts**   | Native Canvas API                  |
| **Fonts**    | System Font Stack (`Segoe UI`, `Poppins`) |

> Zero external dependencies — no Bootstrap, Tailwind, React, or any other framework.
